QED-C Application-Oriented Benchmarks

Version 2.0 — The repository has been restructured into two packages: qedclib (execution library) and qedcbench (benchmarks). For the previous structure, use branch master-260411-v1.2.2.

The QED-C Application-Oriented Benchmarks provide a suite of quantum computing benchmark programs that measure the performance of quantum computers on practical, application-relevant tasks. The benchmarks are maintained by the Quantum Economic Development Consortium (QED-C) Technical Advisory Committee on Standards and Benchmarks.

Documentation Overview

Documentation is organized into the following sections, each addressing a specific need.

Quick Start

The Quick Start is for first-time users of the benchmark suite. It provides a concise walkthrough of installation, running your first benchmark, and understanding the output. A perfect way to get up and running quickly.

    Go to the Quick Start guide.

User Guide

The User Guide is the definitive reference for all features of the benchmark suite. Here you will find information about the benchmark architecture, execution options, metrics interpretation, supported platforms, and how to use qedclib as a standalone library.

    Go to the User Guide.

Benchmarks

The Benchmarks section describes each of the benchmark applications available in the suite, including their purpose, algorithm, and key parameters. Ready-to-use programming examples are provided as Python scripts and Jupyter notebooks showing how to run benchmarks from code, execute on different hardware backends, and integrate external circuit sources.

    Go to the Benchmarks overview.

Release Notes

The benchmark suite is continually evolving, with issues fixed and enhancements made to add features. This section presents a record of changes made to each released version.

    Go to the Release Notes.

Known Issues

As with most complex and evolving software, the benchmark suite has several issues you may want to be aware of. It helps to know what to expect when working with different platforms and execution modes.

    Go to the Known Issues.

About

The About section provides background information about the project, its contributors, the repository structure, and related resources.

    Go to the About section.

Setup Guides

Detailed instructions for setting up specific quantum computing platforms (Qiskit, CUDA-Q, Cirq, Braket, Ocean) are available in the setup guides.

    Go to the Setup Guides.
